# SeaTalk1-to-SignalK-SensESP-v3
Robust SeaTalk 1 to Signal K bridge for ESP32 and SensESP v3. Connects legacy Raymarine instruments (ST5000+, ST60 series) using a hardware-rectified optocoupler circuit and reliable 8-bit gap-based synchronization. Fully compatible with modern SensESP 3.x. Includes high-precision formulas for heading, rudder, wind, and depth
